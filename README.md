# PoE_Part_2
# Overview
# This repository contains **Part 2** of the Dagbreek Spar website project.  
# The focus of this part is on **CSS styling and responsive design**, building on the HTML structure from Part 1.  

# Key features implemented:
# External stylesheet (`css/style.css`) linked to all pages  
# Base styles with reset and CSS variables  
# Typography scale for headings, paragraphs, and lists  
# Layout using Flexbox/Grid for desktop, stacked layout for mobile  
# Visual styling with colors, buttons, and shadows  
# Pseudoâ€‘classes for hover/focus/active states  
# Responsive design with breakpoints for mobile, tablet, and desktop  
# Responsive images with `max-width: 100%` and `srcset`  
# Accessibility improvements (labels, contrast, focus outlines)
